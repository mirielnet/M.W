Lisna

Miriel (@miriel.net) 開発の OSS Discord ボット

Lisna は、100サーバーへの導入を目指しています!!

Commands

/level - あなたのレベルを表示します。
/level-server - サーバーのレベルランキングを表示します。
/level-settings - サーバーのレベル設定を行います。
/translate - 指定されたテキストを翻訳します。
/kuronekoyamato - クロネコヤマトの追跡番号を調べて表示します。
/help - すべてのスラッシュコマンドとその説明を表示します。
/timeout - 指定したユーザーをタイムアウトします。
/kick - 指定したユーザーをキックします。
/ban - 指定したユーザーをバンします。
/dice - チンチロリンで遊びます。
/ping - 反応速度を確認します。
/play - YouTubeまたはSoundCloudの音楽を再生します。
/queue - 現在の再生キューを表示します。
/chu-profile - CHUNIREC(CHUNITHM)のプレイヤープロフィールを取得します。
/autorole_set - 自動ロールを設定します。
/autorole_update - 自動ロールを変更します。
/autorole_remove - 自動ロールの設定を解除します。
/panel - 指定されたロールパネルを作成します。
/botinfo - BOTの実行環境を確認できます。
